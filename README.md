# 🎮 Player Elimination Game (Circular Linked List)

This project is a Java-based simulation game using a Circular Linked List.

Players are randomly selected from a name list and eliminated step by step until one winner remains.

---

## 🚀 Features

✔ Random player selection  
✔ Circular linked list structure  
✔ Step-by-step elimination  
✔ Random gameplay mechanics  
✔ Winner detection  

---

## 📁 Files

- Problem1.java  
- names.txt  

---

## ▶️ How to Run

1. Run Problem1.java  
2. Enter number of players  
3. Watch the elimination process  

---

## 🎯 Game Logic

- Players are selected randomly from a name list  
- Each round:
  - A random player is chosen  
  - The player loses one letter from their name  
- If a player has no letters left → eliminated  
- Last remaining player → winner
